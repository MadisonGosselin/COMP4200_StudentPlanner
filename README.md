#Study Planner App

Overview
The app is a Study Planner that allows users to:
- Log in
- Manage tasks
- View task details
- Track study progress

---

• Current Features
- Login Screen (basic validation)
- Home Screen (MainActivity)
- Add/Edit Task Screen
- Task Details Screen
- Local database structure using Room

---

• Project Structure:
- LoginActivity
- MainActivity
- AddEditTaskActivity
- TaskDetailActivity

• Database (Room)
- Task (Entity)
- TaskDao (DAO)
- TaskDatabase

• Layouts
- activity_login.xml
- activity_main.xml
- activity_add_edit_task.xml
- activity_task_detail.xml

---

Team Workflow
- `main` branch = base project
- Each member creates their own branch
- Do NOT push directly to main
- Use pull requests to merge changes
